Code Tag & Escape Characters

Use <code> for inline code and <pre> for preformatted blocks.

Inline code

Write `const x = 5;` as <code>const x = 5;</code> — it shows inline and monospace.

Code block

// A small code block example
function greet(name) {
  return 'Hello, ' + name;
}

Escape characters (showing HTML)

To show HTML tags literally, escape them using HTML entities:

  • &lt; for <
  • &gt; for >
  • &amp; for &

Example: type &lt;div&gt;Hi&lt;/div&gt; to display <div>Hi</div>.
