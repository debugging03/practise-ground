Put the same header/nav in every page (or use server includes) so users can move around easily.

Relative vs absolute links

Relative: about.html — good for same website. Absolute: https://example.com/about — links anywhere.

Example: link to a subfolder

If pages are in a folder, use pages/about.html. From a subfolder up one level use ../index.html.
