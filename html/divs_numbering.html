Div Tag and List Numbering

What is a <div>?

<div> is a generic container — use it to group content and style with CSS.

Example wrapper:

I am inside a div

Change OL numbering format

Use the type attribute or CSS list-style-type.

HTML attribute examples:

  1. 1 (default)
  1. A
  2. B
  1. a
  2. b
  1. I
  2. II

CSS example (more flexible):

  1. Upper Roman
  2. Second
  1. alpha
  2. beta
